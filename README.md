# STALKER Тест-тренажер

Проект готов к публикации через GitHub Pages.

## Что публиковать

- `index.html` — стартовая страница для GitHub Pages
- `Тест монстры.html` — сам тренажер

## Быстрый деплой через веб-интерфейс GitHub

1. Создайте новый публичный репозиторий на GitHub.
2. Загрузите в него файлы из этой папки.
3. Откройте `Settings -> Pages`.
4. В `Source` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/ (root)`.
6. Нажмите `Save`.
7. Через 1-2 минуты сайт будет доступен по ссылке:
   `https://<username>.github.io/<repo-name>/`
